# 🗺️ Algorithm Visualizer

A pathfinding algorithm visualizer built using React. This project was built for learning purposes; I focused on learning react and implementing a front-end that looks visually appealing.

## 📫 Technologies

- React.js
- HTML
- CSS

## 📖 Features

What you can do in this program:
- **Adding Wall** - Click and drag over endpoints to create nodes that algorithms cannot pass
- **Adjustable Endpoints** - Click and drag either endpoints to move to desired location
- **Animation Types** - Selected by the dropdown on the Navbar
- **Weighted Nodes** - To simulate that some nodes takes longer are to pass than another

## 🚥 Running the project

Steps to run this project:
1. Clone this repository
2. Run `npm install` to install the required depedencies
3. Run `npm start` to start the program on your localhost
